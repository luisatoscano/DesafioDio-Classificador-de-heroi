# Desafios de Lógica de Programação em JavaScript — DIO

Este repositório contém os projetos desenvolvidos durante os desafios práticos de **Lógica de Programação em JavaScript** da **Digital Innovation One (DIO)**. O objetivo é aplicar os conceitos fundamentais da programação, praticar o raciocínio lógico e iniciar a construção de um portfólio no GitHub.

Os desafios abordam o uso de variáveis, operadores, estruturas condicionais, laços de repetição, funções, classes e objetos, utilizando a linguagem JavaScript.

O primeiro desafio consiste na classificação do nível de um herói com base na quantidade de experiência (XP). De acordo com o valor informado, o herói é classificado em níveis como Ferro, Bronze, Prata, Ouro, Platina, Ascendente, Imortal ou Radiante.

O segundo desafio é uma calculadora de partidas ranqueadas, onde uma função recebe a quantidade de vitórias e derrotas de um jogador, calcula o saldo de vitórias e define o nível ranqueado, exibindo uma mensagem com o resultado final.

O terceiro desafio envolve a criação de uma classe genérica que representa um herói de uma aventura, contendo propriedades como nome, idade e tipo. A classe possui um método responsável por executar um ataque, exibindo no console uma mensagem de acordo com o tipo do herói.

Este repositório tem como finalidade reforçar os fundamentos da lógica de programação, introduzir conceitos de programação orientada a objetos e servir como base para projetos futuros mais avançados.

Projeto desenvolvido na plataforma **Digital Innovation One (DIO)**.
